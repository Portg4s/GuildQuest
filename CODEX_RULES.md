# CODEX_RULES

Regles permanentes du projet GuildQuest :

- Repondre en francais dans les rapports.
- Travailler directement a la racine du projet deja ouvert.
- Ne pas creer de sous-dossier projet `GuildQuest`, `guildquest` ou `app`.
- Ne pas push Git.
- Ne pas commit sans demande explicite.
- Ne pas ajouter d'assets proteges ou sous licence.
- Garder `public/private-assets/` ignore par Git.
- Garder `src/data/characters/characters.local.ts` ignore par Git.
- Mettre a jour `CODEX_REPORT.md` apres chaque tache.
- Lancer `npm run build` si possible.
- Lancer `npm run lint` si disponible.
- Signaler clairement les erreurs restantes, commandes echouees et points a verifier.

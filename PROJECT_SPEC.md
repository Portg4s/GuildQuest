# PROJECT_SPEC - GuildQuest

GuildQuest est une application personnelle mobile/offline-first qui transforme l'apprentissage du developpement web en experience de guilde RPG/manga.

## Vision

- Nom : GuildQuest
- Type : PWA personnelle mobile
- Mode : offline-first
- Sujet : apprentissage gamifie du developpement web
- Boucle principale : missions, quiz, progression, recompenses
- Progression : XP, niveaux, gemmes, rangs, badges et titres
- Collection : gacha, personnages, raretes, personnage actif
- Contenu : packs JSON extensibles
- V1 : aucun backend, donnees locales uniquement

## Joueur par defaut

- username : leb
- rang initial : Mage de Rang F
- niveau initial : 1
- XP initiale : 0
- gemmes initiales : 200

## Donnees et assets

Le repo ne doit contenir aucun asset protege. Les images privees ou sous licence doivent rester dans `public/private-assets/`, dossier ignore par Git. Les donnees publiques utilisent des placeholders generiques.
